# Rezerva Appointment Management System

Rezerva is a responsive appointment management dashboard built as a frontend web project. It presents an operational interface for clinics, salons, offices, and consultants with dashboard metrics, booking flow, schedule cards, team overview, and interactive appointment creation.

## Features

- Responsive business dashboard layout
- Appointment booking form with live preview
- Interactive schedule and team sections
- Animated cards, toast feedback, and active navigation states
- Built with plain HTML, CSS, and JavaScript

## Technologies

- HTML
- CSS
- JavaScript
