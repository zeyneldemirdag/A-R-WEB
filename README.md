## About The Project

A-R-WEB is a modern one-page aviation website that explains aircraft mechanics, internal systems and national aviation platforms. The project includes a strong hero section, technical SVG diagrams, scroll animations, counter effects and responsive card layouts.

This repository also includes additional portfolio projects:

- **Knowledge Atlas**: an interactive knowledge-map interface for science, technology, future technologies and book notes.
- **Rezerva**: an appointment management dashboard for clinics, salons, offices and consultants.
- **BMI & Blood Pressure Risk App**: a Flutter university project for BMI calculation, blood pressure input and hypertension risk prediction flow.

## Key Features

- Responsive one-page web interfaces
- Glassmorphism, card layouts and scroll animations with modern CSS
- SVG-based diagram for aircraft internal systems
- JavaScript counter animations and reveal effects
- GitHub Pages deployment for static web projects
- Separate project architecture in subfolders
