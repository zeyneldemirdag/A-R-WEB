# BMI & Blood Pressure Risk App

This is a university Flutter project focused on BMI and hypertension risk tracking. The app collects height, weight, systolic blood pressure, and diastolic blood pressure values, calculates BMI-related metrics, sends risk inputs to a prediction API, and stores local history for follow-up.

## Features

- BMI calculation from height and weight
- Blood pressure input flow
- Hypertension risk result screen
- Local history using shared preferences
- API service prepared for `/predict` endpoint integration
- Flutter UI with custom input, metric, and risk progress widgets

## Technologies

- Flutter / Dart
- HTTP API integration
- Shared Preferences
- Mobile-first UI
